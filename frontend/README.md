# Frontend

Aplicación React + Vite para visualizar el panel de App BiT 64.

## Desarrollo

```bash
cd frontend
npm install
npm run dev
```

## Notas

- El frontend consume la API del backend en `http://localhost:8000` por defecto.
- La carpeta `node_modules/` no debe subirse al repositorio.
- El servicio web se registra automáticamente desde `src/main.tsx`.
